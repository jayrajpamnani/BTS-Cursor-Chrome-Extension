# BTS Cursor - Chrome Extension

Customize your browser cursor with official BTS logos. Features smart detection that switches between black and white logos based on the website's background color to ensure best contrast.

[**Get it on Chrome Web Store**](https://chromewebstore.google.com/detail/bts-cursor-customizer/cadgnckopmbomidpncdfaibifgfeopkf)

## ✨ Features
- **Smart Auto Mode**: Automatically adapts logo color (Black for light backgrounds, White for dark backgrounds)
- **Manual Modes**: Lock the cursor to White or Black
- **Universal Compatibility**: Works seamlessly across all websites and handles dynamically changing content

## 🛠️ Installation
1. Visit `chrome://extensions/` in Chrome
2. Enable "Developer mode" (top right)
3. Click "Load unpacked" and select the extension folder
4. Pin the extension and choose your desired cursor style

## 🔒 Privacy & Security
- **100% Local**: All settings are stored locally. No data collection or tracking scripts.
